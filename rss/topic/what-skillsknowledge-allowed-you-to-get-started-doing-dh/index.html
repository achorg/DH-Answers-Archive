Learning some basic HTML/CSS for web authoring is also beneficial as a tool for teaching the principles of markup languages. What these have in common is that they provide the beginner with instant (or near instant) gratification that they produced something. Once they begin to run up against (and become conscious of) some of the limitations of Wordle and markup for the web, they can start thinking about more powerful tools like MALLET or more complext markup languages like TEI/XML.
