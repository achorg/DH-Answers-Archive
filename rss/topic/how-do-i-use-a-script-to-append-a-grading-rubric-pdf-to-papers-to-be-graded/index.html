It then passes it off to the Automater join script with a few options (creates a new file with '-joined' in the name). </p> <p>I created a quick gist that essentially does the same thing in Ruby with the option parser that you can get at <a href="https://gist.github.com/886814" rel="nofollow">https://gist.github.com/886814</a>. Essentially you can copy the contents in to a file named 'append.rb' somewhere on your computer. In my test, the directory layout is like this:</p> <p>course_name/append.rb<br /> course_name/pdf_files/<br /> course_name/rubric.pdf</p> <p>Inside of course_name, you would run a command like this:</p> <p>ruby append.rb -a rubric.pdf pdf_files/*.pdf</p> <p>This will create a new directory (output by default) and place all the combined files in there (it also opens the directory when the script finishes).</p> <p>If you want to see more stuff going on, you can run this with a -v flag</p> <p>ruby append.rb -v -a rubric.pdf pdf_files/*.pdf</p> <p>This is a little more verbose than the shell version, and can hopefully get you going in the right direction to tailor this to do what you're needing.</p> <p>HTH,</p> <p>Wayne</p> <p>PS.
