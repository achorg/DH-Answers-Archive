The edge list would have the id, source, and target columns, and the node list would have the id, x, and y columns. Then you import the edge list into Gephi, and join the attributes by importing the node list and joining it to the nodes created by the edge list import using the ID column--see here: <a href="https://forum.gephi.org/viewtopic.php?t=1514" rel="nofollow">https://forum.gephi.org/viewtopic.php?t=1514</a>. </p> <p>Once you do that, you should be able to export it to Google Earth.
