Is it worth rolling out translations for introductory pages before the translations for the content are complete?</p> <p>I'm just starting research on this topic (beginning with the W3C's <a href="http://www.w3.org/TR/i18n-html-tech-lang/#ri20040808.100519373">Internationalization Best Practices: Specifying Language in XHTML &amp; HTML Content</a>) but would love to hear other's experiences. This isn't a question as much about the technical implementation, though I'd love to hear about that too if you want to share. :) </p>
