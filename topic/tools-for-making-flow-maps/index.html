    Other, more powerful mapping tools like ArcGIS and QGIS might be good options for publication quality graphics. For a tool that includes web hosting and interactivity and doesn't require programming, perhaps Tableau.

    If you have an interest in non-map flow diagrams, I've seen people use chord diagrams to show movement between locations (US State migration example); those can be created manually with JavaScript in d3 or perhaps automatically with Circos.
