    If you can get the list in the form of a table, you can use ZeeMaps (http://www.zeemaps.com) for this purpose.

    For example, in this case, you'll need a table with columns Name, Address and then upload it to a map on ZeeMaps using a CSV file or just copying-and-pasting the table.

    Let us know at support at zeemaps dot com if we can help.

  3. Peter Organisciak

    You can quickly get a map using a spreadsheet from Google Docs. Paste your data into a column, select that column, and go to Insert>Gadget then Map.

    That will give you the map, you can then also embed it into other pages but choosing Publish Gadgets in the map's setting.

    Here's a sample with your data: https://spreadsheets.google.com/ccc?key=0AgnmwOU2IhTjdDhpY1NNNWJuNWtVd1hFTmJwSUlCbnc&hl=en

    The basic approach I'd use is one of "web scraping" -- extracting the data you need from the HTML page, and transforming the data into a format that you can feed to your mapping application.

    My web scraper of choice is eXist (http://www.exist-db.org), since it lets you crawl through pages using XQuery and XPath - super useful tools for dealing with the web or any XML data you might have. I wrote a script that outputs the addresses as a table. I chose a table format, naturally, since the above-linked Zeemaps accepts data in the form of a copied and pasted table. Pasting this table into Zeemaps, tada, here's your map:

    http://www.zeemaps.com/map?group=212304&x=-96.8555&y=34.7416&z=13
