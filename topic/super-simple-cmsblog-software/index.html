    Not sure what "archives" pages are so this might be no help at all...but here's a page that gives code for building a PHP-based RSS feed creator that pulls content from a MYSQL database. Guess you'd have to embed your 'archive pages' content in a MySQL database.

    http://www.webreference.com/authoring/languages/xml/rss/custom_feeds/

    If you want to run something from your desktop...and if that desktop is a Mac, you can use Sandvox. It lets you create & publish a website to a server and comes with the ability to generate RSS feeds from pages (or groups of pages). I use this package all the time to create quick websites of an ephemeral nature like this one and this one.
