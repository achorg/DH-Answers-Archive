    Or you can generate your own ordered CSV listing in a single--though horrible--line of Ruby:

    1

    IO.read('roland.txt').split(/\W+/).inject(Hash.new(0)){ |h,x| h[x] += 1; h }.sort_by{|k,v| -v}.each{ |a| print "\"#{a[0]}\",#{a[1]}\n" }

    That will print a list of all the words in roland.txt, John O'Hagen's translation of the Song of Roland:

    Replying to @Ben Brumfield's post:

    If you have the file in plain text, you can use Unix utilities.

    tr -sc '[A-Z][a-z]' '[12*]' < input.txt | sort | uniq > output.txt

    Which is to say, read the file input.txt, putting each word onto one line, sort the words alphabetically, and remove any duplicates, saving the file to output.txt.

    P.S. I cannot for the life of me get the forum to stop stripping characters. The third set of brackets should contain "backslash zero one two star" in numeric characters, not 12*.
