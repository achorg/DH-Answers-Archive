    You can use Atlas TI (A-Docs function: synchronize the track with an empty rtf file, and access edition mode) to transcript directly 'inside' a CAQDA package, but the greatest software out there is F4.
    Go to their website: http://www.audiotranskription.de/english/f4.htm and download it for free. It accelerates transcribing by adding the spooling interval, the function that automatically rewinds your interview a bit (you set how many seconds) to allow yourself to get back to the context every time you pause the track. Second thing it does is synchronization (either automatic, with every ENTER hit or manual) that can be exported to Atlas TI or MaxQDA. It's quite simple in use but magnificent, integrating the player functions & adding those simple tricks makes your transcribing hell lot more effective.
    If you want to edit an audio file beforehand, an easy to use but still powerful & free application would be Audacity (http://audacity.sourceforge.net/).

    A few Mac packages that make it a bit easier to do manual transcriptions (as opposed to something like Scribe which claims to do it automatically (assuming it 'knows' your voice and it's your voice that it is transcribing)).

    Haven't tried this one but it looks pretty nice: Transcriva

    I've actually used this one to transcribe the remarks that Cliff Lynch gave at our OA Week event and it's pretty nice for a "free" program: Express Scribe (has Windows version too)

    And finally, Scrivener 2.0 has a transcribe function. Not as nice as Express Scribe but at least it is built right in.
