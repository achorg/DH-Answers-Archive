    require 'rubygems'
require 'libxml'
 
include LibXML
 
# process a directory recursively
def process_directory(directory)
  Dir.new(directory).each do |file|
    full_file = File.expand_path(file, directory)
    if file[0,1] == '.' # skip dot files
      next
    elsif File.directory? full_file
      process_directory full_file
    elsif File.extname(file) == '.xml'

    end
  end

    Replying to @Stéfan Sinclair's post:
    Rather than a custom script, it might be simpler to just use xmllint (the command line tool bundled with libxml2) for this. It can handle multiple files and if you use the "--valid" switch it will check against included DTDs. You could combine this with libxml's catalog support to cache local copies of the relevant DTDs.
